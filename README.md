# Приложение для обработки изображений (вариант 7)

Десктоп-приложение на Python для загрузки и обработки изображений: выделение цветовых каналов, негатив, вращение и рисование фигур.

## Требования

- Python 3.10+
- Веб-камера (опционально, для съёмки снимка)

## Установка и запуск из исходного кода

### 1. Клонирование репозитория

```bash
git clone https://github.com/Badroykus/MoiaPractikaPoYchebe2518V7.git
cd MoiaPractikaPoYchebe2518V7
```

### 2. Создание и активация виртуального окружения

**Windows (PowerShell):**

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

**Windows (cmd):**

```cmd
python -m venv .venv
.venv\Scripts\activate.bat
```

**Linux / macOS:**

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Запуск приложения

```bash
python main.py
```

## Функциональность

### Общий функционал

| Действие | Описание |
|----------|----------|
| **Загрузить изображение** | Выбор файла PNG или JPG с обработкой ошибок |
| **Снимок с веб-камеры** | Захват кадра с подсказками при ошибке подключения |
| **Показать оригинал** | Возврат к исходному изображению |
| **Цветовые каналы** | Отображение красного, зелёного или синего канала |

### Дополнительные пункты меню «Обработка»

| Пункт | Описание |
|-------|----------|
| **Негативное изображение** | Инверсия цветов |
| **Вращение изображения** | Поворот на угол, введённый пользователем |
| **Нарисовать круг** | Красный круг по координатам центра и радиусу |

Статусная строка внизу окна отображает результат каждого действия.

## Решение проблем с веб-камерой

Если камера не подключается:

1. Проверьте, что камера подключена и видна в системе.
2. Закройте программы, которые могут занимать камеру.
3. Разрешите доступ к камере в настройках Windows.
4. Обновите драйверы в «Диспетчере устройств».
5. Перезагрузите компьютер и повторите попытку.
