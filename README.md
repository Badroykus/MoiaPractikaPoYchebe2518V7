Загрузить программу удобным способом
Открыть программу любым удобным способом, например через пачарм

В окне программы весь доступный функционал доступен сверху
Открыв окна вы сможете им воспользоваться
Эта программа умеет 
  Делать негативное изображение
  Вращать изображение
  Рисовать круг на изображении
  Работать с синим, красным и зеленым каналом
Также программа может делать снимок с веб камеры
Программа устойчива к некорректному вводу
